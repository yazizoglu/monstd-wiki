# Poison

## Overview

Poison damage applies a damage-over-time (DoT) effect that continues after the projectile lands. It is the primary counter to regenerating enemies and excels on long path sections where the DoT has time to deal its full value.

---

## Towers

- [[Poison Tower]] — single target, applies DoT on hit

---

## Matchup Table

| vs Enemy | Effectiveness |
|---|---|
| Poison-resistant | Reduced DoT |
| Regenerating | **DoT outpaces regeneration** |
| Armored | Standard |
| Frost-resistant | Standard |
| No resistance | Standard DoT |

---

## Design Notes

- DoT should not stack from multiple Poison Towers — only the strongest active DoT applies
- The "anti-regen" role should be communicated clearly in the UI
- Poison has no strong element counter (unlike Fire/Frost), giving it broader utility but lower peak output
