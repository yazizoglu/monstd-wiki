# Beginner Guide

Welcome to MonsTD. This guide covers core mechanics and the most common early mistakes.

---

## Core Loop

1. **Place towers** on available tiles before the wave starts
2. **Start the wave** with the horn button
3. Enemies travel along the fixed path toward your base
4. Each enemy that reaches the exit costs **1 life** (bosses cost more)
5. Earn gold from kills and wave bonuses — spend it on new towers and upgrades
6. Survive all 25 waves to complete the level

---

## First Build Advice

Start every run with:

1. **[[Arrow Tower]]** — place 2–3 at different coverage angles immediately
2. **[[Fire Tower]]** — add 1–2 at the densest chokepoint by wave 3
3. **[[Frost Tower]]** — add 1 upstream of your main damage cluster by wave 5

This three-tower mix handles the first 10 waves comfortably on Normal difficulty.

---

## Gold Management

- Don't spend all gold on a single tower type
- Save ~60 gold before each mini-boss wave (5, 10, 15, 20) to respond to what you see
- Upgrade existing towers before placing new ones once you have 3–4 towers

---

## Common Mistakes

| Mistake | Better Approach |
|---|---|
| Only using Arrow Tower | Add Fire Tower for swarms |
| Ignoring Frost Tower | 1 Frost Tower upstream doubles your damage output |
| Selling towers too early | Only sell to immediately replace with something better |
| Skipping wave bonuses | Let the wave timer run out only if you need placement time |
| Building identical towers | Resistances make diversity mandatory from Level 2 onward |

---

## Boss Waves

- Bosses appear at waves **5, 10, 15, 20, and 25**
- Mini-bosses cost **3 lives** on escape; the final boss costs **10 lives**
- The final boss (wave 25) also **cancels the wave bonus gold** if it escapes
- Prioritize finishing the boss — don't let it escape even if you have to sell and re-place
